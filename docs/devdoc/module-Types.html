# Point

A track point object. Carries the data for this point and for the previous section.

Properties:
Name Type Description
lat number

Latitude of point
lon number

Longitude of point
date number

Timestamp of point
ignore boolean

If run was paused and this section should therefore be ignored
duration number

Duration of section, in seconds
sumDuration number

Total duration until point, in seconds
distance number

Length of section, in meters
sumDistance number

Total distance until point, in seconds
elev number

Elevation of point, in meters
elevDiff number

Change in elevation over section, in meters
sumElevGain number

Total elevation gain until point
incline number

Incline of section, represented as a fraction
pace number

Speed of section, in km/h
sumPace number

Weighted sum of pace until point, in km/h * s
gap number

Grade adjusted speed of section, in km/h
sumGap number

Weighted sum of gap until point, in km/h * s
hr null | number

Heart rate of point, in bpm, may not exist
sumHr number

Weighted sum of hr until point, in bpm * s, 0 if hr doesn't exist
cad null | number

Cadence of point, in spm, may not exist
sumCad number

Weighted sum of cad until point, in spm * s, 0 if cad doesn't exist
temp null | number

Temperature of point, in degrees C, may not exist
sumTemp number

Weighted sum of temp until point, in C * s, 0 if temp doesn't exist
